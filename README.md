# Versera

Welcome to Versera - a **community-driven, free, developer-friendly** music catalogue sharing platform which allows you to get access to the lyric of the songs you like!

Versera provides:
- Community-driven music catalogue that anyone can contributes
- Plain, line-synced or word-by-word lyrics are available
- Lyrics verified by our `Stewards` and `Specialists`
- Use our catalogue in your app for free
- (Artist Only) Manage, edit and lock your arist profile, albums, tracks and lyrics directly on the platform without any fees
- Completely free-to-use API for developers

While most of our things are in private and not open-source, we still provides easy-to-use API for developers with no cost at all.
